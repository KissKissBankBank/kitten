# Padding

Padding utilities.

These CSS utilities are complemented with the [`padding()` JS helper](?path=/story/helpers-padding--page)

## Source

[`assets/stylesheets/kitten/utilities/spacing/_padding.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/spacing/_padding.scss)

## Usage

The reset padding utility is `k-u-padding-none`.

The basic structure of a padding utility class is
`k-u-padding-(direction)-(value)[@(media query)]`.
Here are some examples:

```
<div class="k-u-padding-top-single">…</div>
<div class="k-u-padding-bottom-double">…</div>
<div class="k-u-padding-left-tripleHalf">…</div>
<div class="k-u-padding-horizontal-sextuple">…</div>
<div class="k-u-padding-vertical-noneHalf@l-up">…</div>
<div class="k-u-padding-horizontal-quintupleHalf@xxs-down">…</div>
```

### `direction`

`direction` can have one of these six values

- `top`
- `bottom`
- `left`
- `right`
- `horizontal` (applies to both `top` and `bottom`)
- `vertical` (applies to both `left` and `right`)

### `value`

`value` can be one of these 22 vales

- `none` will be equivalent to `0`
- `noneHalf` will be equivalent to `5px`
- `single` will be equivalent to `10px`
- `singleHalf` will be equivalent to `15px`
- `double` will be equivalent to `20px`
- `doubleHalf` will be equivalent to `25px`
- `triple` will be equivalent to `30px`
- `tripleHalf` will be equivalent to `35px`
- `quadruple` will be equivalent to `40px`
- `quadrupleHalf` will be equivalent to `45px`
- `quintuple` will be equivalent to `50px`
- `quintupleHalf` will be equivalent to `55px`
- `sextuple` will be equivalent to `60px`
- `sextupleHalf` will be equivalent to `65px`
- `septuble` will be equivalent to `70px`
- `septubleHalf` will be equivalent to `75px`
- `octuple` will be equivalent to `80px`
- `octupleHalf` will be equivalent to `85px`
- `nonuple` will be equivalent to `90px`
- `nonupleHalf` will be equivalent to `95px`
- `decuple` will be equivalent to `100px`
- `auto` will be equivalent to `auto`

### `media query`

`media query` is optional and follows the general scheme described in
[Using Utilities: Media Queries](?path=/story/documentation-usage-using-utilities--page#media-queries)
