# Display

Display utilities.

## Source

[`assets/stylesheets/kitten/utilities/display/_display.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/display/_display.scss)

## Usage

Utility to set the items `display` property according to media-queries.

Accepted classes:

- **`k-u-block`** (`display: block;`)
- **`k-u-inlineBlock`** (`display: inline-block;`)
- **`k-u-inline`** (`display: inline;`)
- **`k-u-flex`** (`display: flex;`)
- **`k-u-inlineFlex`** (`display: inline-flex;`)
- **`k-u-grid`** (`display: grid;`)
- **`k-u-inlineGrid`** (`display: inline-grid;`)
- **`k-u-contents`** (`display: contents;`)
- **`k-u-hidden`** (`display: none;`)
