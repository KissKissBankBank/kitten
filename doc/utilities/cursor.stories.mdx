# Cursor

Cursor utilities.

## Source

[`assets/stylesheets/kitten/utilities/_cursor.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/_cursor.scss)

## Usage

Utilities to cursor items (cursor: default, pointer, not-allowed, crosshair).

`k-u-cursor-default`
<Canvas>
<p className="k-u-cursor-default">
  Hello text cursor default!
</p>
</Canvas>

`k-u-cursor-pointer`
<Canvas>
<p className="k-u-cursor-pointer">
  Hello cursor pointer!
</p>
</Canvas>

`k-u-cursor-not-allowed`
<Canvas>
<p className="k-u-cursor-not-allowed">
  Hello text! I'm on the not allowed.
</p>
</Canvas>

`k-u-cursor-crosshair`
<Canvas>
<p className="k-u-cursor-crosshair">
  Hello text! I'm on the crosshair.
</p>
</Canvas>
