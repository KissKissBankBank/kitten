# Colors

You can find source file [here](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/text/_color.scss).

Add class name `k-u-color-font1` to add `color`

<Preview>
  <p className="k-u-color-font1--important">
    Hello color !
  </p>
</Preview>

Or `k-u-color-font2`

<Preview>
  <p className="k-u-color-font2--important">
    Hello color! Still same style ? No, I'm a different color.
  </p>
</Preview>

Or `k-u-color-primary1` 
<Preview>
  <p className="k-u-color-primary1--important">
    Hello color! I'm blue like the sky.
  </p>
</Preview>

Or `k-u-color-background1`
