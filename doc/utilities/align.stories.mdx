# Align

You can find source file [here](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/text/_align.scss).

Utilities to align items according to media-queries (aligns : center, left, right).

`k-u-align-center`

  </p>
</Canvas>

<Canvas>
