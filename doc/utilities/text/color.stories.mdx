# Colors

Text color utilities.

## Source

[`assets/stylesheets/kitten/utilities/text/_color.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/text/_color.scss)

## Usage

Add class name `k-u-color-font1` to add `color`

<Canvas>
<p className="k-u-color-font1">
  Hello color!
</p>
</Canvas>

Or `k-u-color-font2`
<Canvas>
<p className="k-u-color-font2">
  Hello color! Still same style ? No, I'm a different color.
</p>
</Canvas>

Or `k-u-color-primary1`
<Canvas>
<p className="k-u-color-primary1">
  Hello color! I'm blue like the sky.
</p>
</Canvas>

Or `k-u-color-background1`
<Canvas style={{ backgroundColor: 'black' }}>
<p className="k-u-color-background1">
  Hello color! I'm white as white text on a black background!
</p>
</Canvas>

Or `k-u-color-error`
<Canvas>
<p className="k-u-color-error">
  Hello color! I'm red with anger.
</p>
</Canvas>

Or finally `k-u-color-valid`
<Canvas>
<p className="k-u-color-valid">
  Hello color! I'm green like a well-watered lawn.
</p>
</Canvas>
