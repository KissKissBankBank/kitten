# Align Self

These self-alignment utilities will only be useful inside of a Flexbox or a CSS
Grid container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/align/_align.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/align/_align.scss)

## Usage

Utilities to align items according to media-queries (aligns : normal, stretch,
center, start, end).

`k-u-alignSelf-normal`
<Canvas className="has-overrides show-flex">
<div className="k-u-flex">
  <p className="k-u-margin-right-double">Default for comparison</p>
  <p className="k-u-alignSelf-normal">
    Normal alignment (defaults to stretch).
  </p>
  <p className="k-u-margin-left-double">Default for comparison</p>
</div>
</Canvas>
