# Align self

Align-self utility to be used with a flexbox container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/flex/_align-self.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_align-self.scss)

## Usage

Utilities to align self according to media-queries.

The class name has the form `k-u-flex-alignSelf-{value}`

Accepts values:

- `normal`
- `stretch`
- `center`
- `start`
- `end`
