# Flex wrap

Flex wrap utility to be used with a flexbox container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/flex/_flex-wrap.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_flex-wrap.scss)

## Usage

Utilities to set items wrap according to media-queries.

The class name has the form `k-u-flex-wrap-{value}`

Accepts values:

- `wrap`
- `reverse`
- `nowrap`
