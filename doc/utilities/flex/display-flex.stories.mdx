# Display Flex

Two utilities to display a block as flex.

## Source

[`assets/stylesheets/kitten/utilities/flex/_display-flex.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_display-flex.scss)

## Usage

Two class names are available:

- `k-u-flex`
- `k-u-inlineFlex`

## Examples

`k-u-flex`
<Canvas className="has-overrides show-flex">
<div className="k-u-flex">
  <p>
    First block item
  </p>
  <p>
    Second block item
  </p>
  <p>
    Third block item
  </p>
</div>
</Canvas>
