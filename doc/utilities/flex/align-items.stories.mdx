# Align items

Align-items utility to be used with a flexbox container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/flex/_align-items.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_align-items.scss)

## Usage

Utilities to align items according to media-queries.

The class name has the form `k-u-flex-alignItems-{value}`

Accepts values:

- `normal`
- `stretch`
- `center`
- `start`
- `end`
