# Justify content

Justify-content utility to be used with a flexbox container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/flex/_justify-content.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_justify-content.scss)

## Usage

Utilities to justify flexbox content according to media-queries.

The class name has the form `k-u-flex-justifyContent-{value}`

Accepts values:

- `normal`
- `center`
- `start`
- `end`
- `left`
- `right`
- `sa` (for `space-around` property)
- `sb` (for `space-between` property)
- `se` (for `space-evenly` property)
- `stretch`
