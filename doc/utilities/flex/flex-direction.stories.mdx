# Flex direction

Flex direction utility to be used with a flexbox container, like `k-u-flex` or `k-u-flex-inline`.

## Source

[`assets/stylesheets/kitten/utilities/flex/_flex-direction.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/flex/_flex-direction.scss)

## Usage

Utilities to set items direction according to media-queries.

The class name has the form `k-u-flex-direction-{value}`

Accepts values:

- `row`
- `rowReverse`
- `column`
- `columnReverse`
