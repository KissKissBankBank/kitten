# Cursor

Display utilities to display an element as an inline block.

## Source

[`assets/stylesheets/kitten/utilities/display/_block.scss`](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/utilities/display/_block-inline.scss)

## Usage

Utility to display items as inline-blocks according to media-queries.
Can be used with the [hidden](?path=/docs/documentation-css-utilities-display-hidden--page) utility.

The class name has the form `k-u-block-inline`.

**For better effect with the Hidden utility, when not using the utilities.css stylesheet, the `k-utilities-block` and `k-utilities-blockInline` mixins have to be imported after the `k-utilities-hidden` util.**

```scss
@include k-utilities-hidden;
@include k-utilities-block;
@include k-utilities-blockInline;
```
