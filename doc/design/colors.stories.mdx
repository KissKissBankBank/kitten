# Colors

Kitten propose un lot de couleur (iso Figma) que vous pouvez appeler dans le code de differentes manieres:

### Exemples

```css
.p {
  background: var(--color-grey-100);
  color: var(--color-primary-500);
}
```
ou 
```html
<p style={{ backgroundColor: 'var(--color-danger-100)' }}>...</p>`
```

- Avec className : 

```html
<p className="k-u-color-grey-100">...</p>
```

`k-u-color` étant la base sur laquelle vous allez appeler les couleurs puis `primary-100` ou `grey-100` etc etc..

Source : [assets/stylesheets/kitten/tokens/_colors.scss](https://github.com/KissKissBankBank/kitten/blob/master/assets/stylesheets/kitten/tokens/_colors.scss)
