# Using Utilities

## Importing

```
@import '@kisskissbankbank/kitten/src/stylesheets/utilities';
```

## Using the utilities

### Media Queries in class names

Equivalent to `@media (min-width: ___) {}`

- `@xxs-up`
- `@xs-up`
- `@s-up`
- `@m-up`
- `@l-up`
- `@xl-up`


Equivalent to `@media (max-width: ___) {}`

- `@xxs-down`
- `@xs-down`
- `@s-down`
- `@m-down`
- `@l-down`
- `@xl-down`
