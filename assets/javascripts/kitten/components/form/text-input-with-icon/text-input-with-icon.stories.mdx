## Usage

#### Default

```jsx
<TextInputWithIcon />
```

#### Size

```jsx
<TextInputWithIcon size="tiny" />
```

#### State

### Default props from `TextInput`

```jsx
<TextInputWithIcon valid />
<TextInputWithIcon error />
<TextInputWithIcon disabled />
```

### With icon

```jsx
<TextInputWithIcon icon={<MyIcon />}>
```

### Icon position

```jsx
<TextInputWithIcon icon={<MyIcon />} iconPosition="right">
```
