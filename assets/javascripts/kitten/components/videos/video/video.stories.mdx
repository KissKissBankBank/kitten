# Video

## Import

```js
import { Video } from '@kisskissbankbank/kitten'
```

## Usage

<Canvas>
  <Story name="Video" story={stories.Default} />
</Canvas>

You can use all `<video />` attributes.

```js
<Video src="…" />
```

#### Loader

```js
<Video src="…">
  <Video.Loader>
    <LoaderIcon />
  </Video.Loader>
</Video>
```
